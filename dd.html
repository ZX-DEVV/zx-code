Modern CSS Section Example

How to Use CSS in a Section

CSS can be used to style sections by applying styles to the <section> tag directly or by assigning it a class or ID. Here are some key tips:

  • Use padding to control spacing inside the section.
  • Apply background-color to highlight the section.
  • Use box-shadow for a modern, lifted look.
  • Adjust border-radius to give rounded corners.

Example Code:

section { padding: 20px; background-color: #101010; border-radius: 8px; box-shadow: 0 4px 6px rgba(0, 255, 255, 0.3); color: #00ffff; }
